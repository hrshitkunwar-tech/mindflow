---
name: Mindflow
description: Collects saved content and bookmarks from social media platforms, organizes them using AI, and generates searchable mind maps for fast knowledge discovery and insight generation.

capabilities:
  - Import and unify saved posts, bookmarks, and links from multiple platforms
  - Automatically cluster and categorize knowledge using AI
  - Provide semantic search across all saved content
  - Generate structured mind maps for visual exploration
  - Surface key insights and themes from stored information

when_to_use:
  - Retrieve previously saved knowledge quickly
  - Organize scattered bookmarks into structured understanding
  - Explore ideas visually through mind maps
  - Build a personal or team knowledge memory layer

usage_examples:
  - Find all saved content about AI agents
  - Create a mind map from my startup research bookmarks
  - Summarize themes from my saved productivity posts
