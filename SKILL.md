# Mindflow

## Description
Mindflow is an AI-powered system that collects saved content and bookmarks from social media platforms, organizes them into structured knowledge, and presents them in a searchable mind-map interface for fast discovery and insight.

## Capabilities
- Import saved posts and bookmarks from social platforms
- Categorize and cluster knowledge automatically
- Provide semantic search across saved content
- Generate visual mind maps for exploration

## Usage
Use this skill to retrieve, organize, and explore saved knowledge across platforms in a unified interface.
